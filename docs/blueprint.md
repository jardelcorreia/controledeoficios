# **App Name**: Controle de Ofícios

## Core Features:

- App Navigation: Implements a sidebar navigation with links to Dashboard, Oficios, History, and Configurations.
- Dashboard Overview: Displays key metrics and recent activity at a glance.
- Oficios Management: Manages the listing, creation, and editing of official documents.
- Oficio Details: Provides a detailed view of individual 'oficio' records.
- Numbering Configuration: Enables configuration of numbering prefixes, suffixes, and base year.
- Action History: Logs all actions performed within the application for audit trails.

## Style Guidelines:

- Primary color: Slate blue (#718096) to convey a sense of professionalism and organization, avoiding clichés of corporate design while hinting at official documentation.
- Background color: Light gray (#F7FAFC), providing a clean and neutral backdrop that allows the slate blue primary to stand out, and contributing to readability.
- Accent color: Sky blue (#90CDF4), used sparingly for interactive elements to draw attention without overwhelming the user.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its readability and modern appearance, well-suited for both headlines and body text.
- Lucide-react icons provide a consistent, minimalist aesthetic for navigation and actions.
- Employs a clear separation of navigation (left sidebar with dark background) and content (main area with light background), creating an intuitive user experience.
- App language: PT-BR.
